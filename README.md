# Plasma Boy vs Plantas Monstruo

A browser arena shooter where a hero kid battles evil plants. Built with Phaser 3 + TypeScript, outputs a single HTML file you can just open — no server needed.

## Play it

```bash
npm run build
open dist/index.html
```

## Dev

```bash
npm run dev
```
